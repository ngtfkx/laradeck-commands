A set of useful Laravel artisan commands

## Install

Via Composer

``` bash
$ composer require ngtfkx/laradeck-commands
```

## Usage

``` php
$skeleton = new League\Skeleton();
echo $skeleton->echoPhrase('Hello, League!');
```
